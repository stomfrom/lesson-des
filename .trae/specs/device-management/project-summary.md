# 星维设备管理系统 — 项目报告

## 任务要求

本系统采用 **Vue 3 + Element Plus** 构建前端 SPA，**Express + MySQL** 构建后端 RESTful API，使用 **JWT + bcryptjs** 实现认证授权。系统面向两类用户：**管理员**（拥有全部权限）和**普通用户**（操作权限由管理员单独分配）。实现的功能包括：设备 CRUD、分页排序筛选、设备详情查看、Dashboard 统计看板、CSV 导出、维保提醒标识、多状态筛选、RBAC 权限控制、用户管理、设备生命周期自动流转（可配置月数），覆盖企业级设备管理的完整业务流程。
